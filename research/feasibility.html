Project Features

Statistics Display: The project highlights the statistic that only 4% of foster children graduate college. It uses a timeline covering elementary, middle, high school, college, and post-grad phases.

Interactive Timeline: Starting with 100 colorful icons, as the user scrolls down, icons disappear to show how many advance to the next educational stage, with a decreasing percentage shown over time.

Technologies and Tools

Fullpage.js: For seamless vertical scrolling.

Animate on Scroll and Chart.js: To animate data and text as they come into view.

Typeit.js: To make the text dynamic and engaging.

Adobe Illustrator and Photoshop: To create and design icons.

New Library for Shatter Animation: Looking for a library that can add dynamic effects to the disappearing icons.
