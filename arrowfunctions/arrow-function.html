Arrow Functions

This function is from my first mini studio project. The checkTime() function shows or hides lines of a haiku poem based on the current time of a video. It goes through each time interval specified for the haiku. If the video is at a time within an interval, it shows the corresponding line. If not, it hides the line. This way, the function helps display the right line of the haiku at the right moment of the video.

Function

            function checkTime() {
                for (let i = 0; i < haiku.start.length; i++) {
                    if (haiku.start[i] < myVideo.currentTime && myVideo.currentTime < haiku.stop[i]) {
                        haiku.line[i].className = 'showing';
                    }
                    else {
                        haiku.line[i].className = 'hidden';
                    }
                }
            }
